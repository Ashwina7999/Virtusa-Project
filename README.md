# Weather Forecast Application

Web application for displaying the Current weather, Air pollution data, Forecasting, Maps and Alert Message (if available).  

### Tech stack

* HTML5
* CSS
* BOOTSTRAP
* JSP
* jQuery

### Server Side

* Spring Boot
* Spring Web

### Database

* MySQL
